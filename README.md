# Scores Leadboard

This is a leaderboard app to displays scores submitted by different players, It is used API to get data and post new score.

![screenshot](./game.png)

## Built using

- HTML
- CSS
- JavaScript
- Webpack


## Getting Started

after cloning
```
git clone https://github.com/omarsalem7/My-leaderboard.git
cd My-leaderboard

```
After That
```
npm install
npm start
```
